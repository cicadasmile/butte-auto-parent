# butte-auto

## 1、案例简介

分布式服务的部署是一个复杂的流程，用手动的方式部署显然难度过高，基于Jenkins、Docker、K8S持续集成组件，实现自动化管理源码编译、打包、镜像构建、部署等系列操作；
